# Basic REST API With Express JS (Node JS) & MySQL

#### How to install

- `npm init`
- `npm install express nodemon`
- `npm install mysql2`
- `npm install dotenv`
- `npm install multer`

Shorthand : `npm i express nodemon mysql2 dotenv multer`

#### How to run this app

`npm start`
